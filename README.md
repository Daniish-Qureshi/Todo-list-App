## 🎯 About The Project

**Todo List App** is a lightweight, browser-based task management application. It lets users add tasks, organize them by category (like Personal), and keep track of daily to-dos — all without any backend or database.

This project was built as part of a web development internship at **Codveda Technology** to practice DOM manipulation, dynamic UI rendering, and local state management using vanilla JavaScript.

---

## ✨ Features

- ➕ **Add Tasks** — Quickly add new tasks with a name and category
- 🗂️ **Category Support** — Organize tasks under categories (e.g., Personal)
- 📊 **Task Counter** — Displays total number of tasks dynamically
- 🗑️ **Delete Tasks** — Remove completed or unwanted tasks
- 🧹 **Cancel / Reset** — Cancel task creation with a single click
- 📱 **Responsive Design** — Works smoothly on mobile and desktop
- 🎨 **Clean UI** — Minimal and distraction-free interface

## 🚀 Getting Started

### Prerequisites

- Any modern browser (Chrome, Firefox, Edge)
- [VS Code](https://code.visualstudio.com/) + [Live Server Extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) *(optional)*

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Daniish-Qureshi/todo-list-app.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd todo-list-app
   ```

3. **Open in browser**
   - Double-click `index.html`, **OR**
   - Right-click in VS Code → **Open with Live Server**

No build step, no dependencies. Just open and use! 🎉

## ⚙️ How It Works

1. **View Tasks** — On load, all existing tasks are displayed with a task count
2. **Add a Task** — Click the **"Add Task"** button, enter a task name and select a category
3. **Confirm or Cancel** — Hit **Add** to save the task or **Cancel** to dismiss
4. **Task Count Updates** — The counter updates automatically when tasks are added or removed
5. **Delete a Task** — Remove any task directly from the list
